## **Хід роботи**

1. **Ознайомився з теоретичними відомостями про API.**

   **API (Application Programming Interface)** — це набір правил, який дозволяє одній програмі взаємодіяти з іншою.  
   У веб-розробці API використовується для обміну даними між клієнтом (браузером) і сервером.  
   Приклади:  
   - *OpenWeather API* — погода  
   - *The Cat API* — зображення котів  
   - *REST Countries API* — інформація про країни світу

2. **Розглянув REST API та HTTP-методи.**

   REST API працює поверх протоколу HTTP і визначає стандартні методи:
   - `GET` — отримати дані
   - `POST` — створити новий ресурс
   - `PUT` — оновити дані
   - `DELETE` — видалити ресурс  

   Наприклад:  
   `GET https://restcountries.com/v3.1/all` — отримати список усіх країн світу.  
   `GET https://restcountries.com/v3.1/name/ukraine` — отримати інформацію про Україну.

3. **Опис використаного API.**

   Для виконання індивідуального завдання (варіант №3) було обрано **REST Countries API** — безкоштовний сервіс, що надає детальну інформацію про країни світу: назву, столицю, регіон, населення, площу, прапор, мови, валюти тощо.

   - Документація: [https://restcountries.com](https://restcountries.com)
   - Основні запити:
     - `GET https://restcountries.com/v3.1/name/ukraine` — інформація про Україну
     - `GET https://restcountries.com/v3.1/all` — список усіх країн
