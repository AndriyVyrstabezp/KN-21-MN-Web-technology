## **Хід роботи**

1.  **Ознайомився з теоретичними відомостями про бібліотеки.**\
    Дослідив різницю між бібліотекою та фреймворком:

    -   Бібліотека --- це набір інструментів, які викликає програміст.
    -   Фреймворк задає структуру, у межах якої працює програміст.

2.  **Розглянув популярні бібліотеки JavaScript:**

    -   **Axios** --- для виконання HTTP-запитів.
    -   **Chart.js** --- для побудови графіків і діаграм.
    -   **Leaflet.js** --- для створення інтерактивних карт.
    -   **Lodash** --- для роботи з масивами, об'єктами, рядками.
    -   **Moment.js** --- для роботи з датами й часом.

3.  **Створив HTML-сторінку з підключенням бібліотеки Chart.js:**

    ``` html
    <!DOCTYPE html>
    <html lang="uk">
    <head>
      <meta charset="UTF-8">
      <title>Температура у с. Дев’ятники (А.В.)</title>
      <script src="https://cdn.jsdelivr.net/npm/chart.js"></script>
    </head>
    <body style="background-color: #fff8f0; font-family: Arial, sans-serif; text-align: center;">
      <h2 style="color: #b34700;">Температура у с. Дев’ятники протягом тижня (А.В.)</h2>
      <canvas id="myChart" width="400" height="200"></canvas>
      <script src="script.js"></script>
    </body>
    </html>
    ```

4.  **Створив JavaScript-файл `script.js` для побудови лінійного
    графіка:**

    ``` javascript
    const ctx = document.getElementById('myChart').getContext('2d');
    const myChart = new Chart(ctx, {
        type: 'line',
        data: {
            labels: ['Пн', 'Вт', 'Ср', 'Чт', 'Пт', 'Сб', 'Нд'],
            datasets: [{
                label: 'Температура (°C)',
                data: [15, 17, 19, 18, 16, 14, 13],
                borderColor: 'rgba(255, 99, 71, 1)',
                backgroundColor: 'rgba(255, 165, 0, 0.3)',
                pointBackgroundColor: 'rgba(255, 215, 0, 1)',
                borderWidth: 2,
                tension: 0.4,
                fill: true
            }]
        },
        options: {
            responsive: true,
            plugins: {
                legend: {
                    labels: {
                        color: '#333'
                    }
                },
                title: {
                    display: true,
                    text: 'Осінні температури у с. Дев’ятники (А.В.)',
                    font: {
                        size: 18,
                        weight: 'bold'
                    },
                    color: '#b34700'
                }
            },
            scales: {
                y: {
                    beginAtZero: false,
                    title: {
                        display: true,
                        text: 'Температура (°C)',
                        color: '#b34700'
                    }
                },
                x: {
                    title: {
                        display: true,
                        text: 'Дні тижня',
                        color: '#b34700'
                    }
                }
            }
        }
    });
    ```

5.  **Відкрив HTML-файл у браузері.**\
    Графік коректно відобразився --- видно лінійну діаграму температури
    з підписами днів тижня у теплих осінніх кольорах.
