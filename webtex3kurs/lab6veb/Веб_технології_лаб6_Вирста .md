## **Теоретичні відомості**

### **Node.js**

Node.js --- це JavaScript-середовище, яке дозволяє запускати JS на
сервері.\
Особливості: - однопоточність; - неблокуюча модель обробки запитів; -
велика екосистема NPM.

### **Express.js**

Фреймворк, що полегшує розробку серверних додатків: - прості маршрути; -
middleware; - підтримка JSON; - можливість роздавати статичні файли.

### **REST API**

REST --- набір правил побудови веб-сервісів: - використання
HTTP-методів; - унікальні URI ресурсів; - сервер не зберігає стану
клієнта.

### **CRUD**

-   C --- Create → POST\
-   R --- Read → GET\
-   U --- Update → PUT\
-   D --- Delete → DELETE

## **3. Реалізація сервера (app.js)**

У головному файлі налаштовано: - підключення Express; - middleware для
JSON; - middleware для логування; - маршрути `/api/tasks`; - статичні
файли `/public`.

Код:

``` javascript
const express = require('express');
const path = require('path');
const tasksRouter = require('./routes/tasks');

const app = express();
const PORT = 3000;

app.use(express.json());

app.use((req, res, next) => {
  console.log(`[${new Date().toISOString()}] ${req.method} ${req.url}`);
  next();
});

app.use(express.static(path.join(__dirname, 'public')));

app.use('/api/tasks', tasksRouter);

app.listen(PORT, () =>
  console.log(`Сервер запущено на http://localhost:${PORT}`)
);
```

# **Контрольні питання та розгорнуті відповіді**

### **1. Що таке REST API?**

REST API --- це архітектурний стиль взаємодії клієнта і сервера, який
базується на HTTP.\
Його особливості: безстанова взаємодія, унікальні URI ресурсів,
стандартизовані методи запитів.

------------------------------------------------------------------------

### **2. Які HTTP-методи використовуються в CRUD?**

-   **GET** --- отримання даних (Read)\
-   **POST** --- створення нового запису (Create)\
-   **PUT / PATCH** --- оновлення даних (Update)\
-   **DELETE** --- видалення (Delete)

------------------------------------------------------------------------

### **3. Для чого використовується express.json()?**

Для автоматичного перетворення JSON-даних із тіла запиту в
JavaScript-об'єкт.\
Без нього `req.body` буде порожнім.

------------------------------------------------------------------------

### **4. Як створюються маршрути в Express?**

Маршрути створюються через методи:

``` javascript
app.get()
app.post()
app.put()
app.delete()
```

або через `express.Router()` для модульності проєкту.

------------------------------------------------------------------------

### **5. Як підключити статичні файли в Express?**

Через:

``` javascript
app.use(express.static('public'));
```

Після цього файли з папки `public` доступні напряму у браузері.
