## **Теоретичні відомості**

1. **Node.js**  
   Node.js — середовище виконання JavaScript поза браузером, побудоване на рушії V8.  
   Дає змогу використовувати JavaScript для серверної частини застосунків.

2. **HTTP‑сервер**  
   Сервер, який приймає запити клієнтів та формує відповіді (HTML, JSON, файли тощо).  
   У Node.js створюється через вбудований модуль `http`.

3. **Модуль http**  
   Забезпечує створення серверів, роботу з запитами й відповідями.  
   Приклад базового сервера:
   ```javascript
   const http = require('http');
   const server = http.createServer((req, res) => {
     res.end('Hello, world!');
   });
   server.listen(3000);
   ```

4. **Маршрутизація (Routing)**  
   Це обробка різних URL‑шляхів, наприклад `/`, `/about`, `/student`.  
   Визначається через перевірку `req.url`.

5. **Асинхронність**  
   Node.js дозволяє обробляти багато запитів одночасно без блокування потоків, що робить його ефективним для веб‑серверів.

## **Хід роботи (практична частина)**

1. **Встановлення Node.js**  
   Завантажив і встановив Node.js із сайту [https://nodejs.org](https://nodejs.org).  
   Перевірив у терміналі команди:
   ```
   node -v
   npm -v
   ```

2. **Створення структури проєкту**
   ```
   /lab4-server
   ├── server.js
   └── student.json
   ```

3. **Вміст `server.js`**
   ```javascript
   const http = require('http');
   const fs = require('fs');
   const path = require('path');

   const student = {
     name: "Андрій Вирста",
     group: "КН-31",
     message: "Я вчуся працювати з Node.js!"
   };

   const filePath = path.join(__dirname, 'student.json');

   if (!fs.existsSync(filePath)) {
     fs.writeFileSync(filePath, JSON.stringify(student, null, 2), 'utf8');
     console.log('Файл student.json створено.');
   }

   const server = http.createServer((req, res) => {
     console.log(`Запит: ${req.url}`);

     if (req.url === '/' || req.url === '/index') {
       res.writeHead(200, { 'Content-Type': 'text/html; charset=utf-8' });
       res.end('<h1>Головна сторінка</h1><p>Ласкаво просимо до мого Node.js сервера!</p>');
     } else if (req.url === '/about') {
       res.writeHead(200, { 'Content-Type': 'text/html; charset=utf-8' });
       res.end('<h1>Про нас</h1><p>Це лабораторна робота №4 з Node.js</p>');
     } else if (req.url === '/student') {
       res.writeHead(200, { 'Content-Type': 'application/json; charset=utf-8' });
       res.end(JSON.stringify(student, null, 2));
     } else if (req.url === '/student/html') {
       res.writeHead(200, { 'Content-Type': 'text/html; charset=utf-8' });
       res.end(`
         <h1>Інформація про студента</h1>
         <table border="1" cellspacing="0" cellpadding="8">
           <tr><th>Ім'я</th><td>${student.name}</td></tr>
           <tr><th>Група</th><td>${student.group}</td></tr>
           <tr><th>Повідомлення</th><td>${student.message}</td></tr>
         </table>
       `);
     } else {
       res.writeHead(404, { 'Content-Type': 'text/html; charset=utf-8' });
       res.end('<h1>Сторінку не знайдено (404)</h1>');
     }
   });

   const PORT = 3000;
   server.listen(PORT, () => {
     console.log(`Сервер працює на http://localhost:${PORT}`);
   });
   ```

4. **Вміст `student.json` (створюється автоматично)**
   ```json
   {
     "name": "Андрій Вирста",
     "group": "КН-31",
     "message": "Я вчуся працювати з Node.js!"
   }
   ```

5. **Результат запуску**  
   У терміналі після виконання команди:
   ```
   node server.js
   ```
   відображається повідомлення:  
   `Сервер працює на http://localhost:3000`

   При переході за адресами:  
   - `/` — головна сторінка;  
   - `/about` — опис проєкту;  
   - `/student` — JSON‑відповідь із даними;  
   - `/student/html` — таблиця з даними студента.

## **Контрольні питання та відповіді**

**1. Що таке Node.js і для чого він використовується?**  
Node.js — середовище виконання JavaScript поза браузером. Використовується для створення серверних застосунків, API та інструментів розробки.

**2. Який модуль у Node.js використовується для створення HTTP‑сервера?**  
Використовується вбудований модуль `http`.

**3. Що таке req і res у Node.js?**  
`req` — об’єкт запиту клієнта, `res` — об’єкт відповіді сервера.

**4. Як організувати маршрутизацію у простому сервері?**  
Через перевірку значення `req.url` і обробку кожного маршруту окремо.

**5. У чому різниця між форматами text/html та application/json?**  
`text/html` — відправляє HTML‑розмітку, яку браузер відображає як сторінку;  
`application/json` — відправляє структуровані дані у форматі JSON.

**6. Як записати дані у файл у Node.js?**  
За допомогою функції `fs.writeFileSync` або асинхронної `fs.writeFile`.

**7. Яка команда запускає сервер?**  
```
node server.js
```
